# 📊 Interactive Data Visualization Dashboard

EASE THE JOURNEY TO UNDERSTAND YOUR DATA!!\n
Use this INTERACTIVE DATA VISUALIZATION DASHBOARD to see trends, fluctuations, patterns and much more inyour data through your choice of graph plotted with your choice features and filters!\n
Upload datasets, build charts, and save configurations.\n
Try it out!:
https://charts-dashboard.streamlit.app/

## Features
- Upload CSV/Excel files
- Choose chart type, axes, and filters
- View interactive Plotly charts
- Save and regenerate graph configurations
- Export data and configs

## How to Run Locally
```bash
pip install -r requirements.txt
