# 📊 Interactive Data Visualization Dashboard

🚀 **Turning Raw Data into Actionable Insights — Instantly!**

An end-to-end **Interactive Data Visualization Dashboard** built using Streamlit that simplifies exploratory data analysis (EDA) by making it fast, intuitive, and code-free.

## 💡 Overview

In real-world data science workflows, the biggest challenge isn’t building models — it’s understanding the data.

This project is designed to bridge that gap by providing a seamless workflow:

> **Upload → Filter → Visualize → Save → Export**

Instead of writing repetitive Pandas and plotting code, users can interactively explore datasets and generate insights in seconds.

---

## ✨ Key Features

### ⚡ Instant Data Exploration
- Upload CSV or Excel datasets and start analyzing immediately  
- No preprocessing or setup required  

### 🔍 Real-Time Interactive Filtering
- Apply filters dynamically  
- Visualizations update instantly without rerunning code  

### 📈 Dynamic Visualizations
- Supports multiple chart types:
  - Bar  
  - Line  
  - Scatter  
  - Pie  
  - Histogram  
- Flexible axis selection for deeper analysis  

### 📊 Built-in EDA & Insights
- Dataset preview  
- Key metrics (rows, columns, missing values)  
- Quick understanding of dataset structure  

### 🔥 Correlation Heatmap
- Analyze relationships between numerical features  
- Identify patterns and dependencies instantly  

### 📦 Data Export (CSV & Excel)
- Export **filtered and processed datasets**  
- Ready for:
  - Machine Learning models  
  - Jupyter notebooks  
  - Business reporting  

### 💾 Persistent Chart Configurations
- Save visualizations along with filters using SQLite  
- Reload and reuse analysis anytime  

### 🎯 Clean & Intuitive UI
- Built with **Streamlit + Plotly**  
- Designed for both technical and non-technical users

## 🛠️ Tech Stack

- **Frontend/UI:** Streamlit  
- **Data Processing:** Pandas  
- **Visualization:** Plotly  
- **Database:** SQLite  
- **File Handling:** CSV, Excel (openpyxl)

## 🚀 Getting Started

### 1. Clone the Repository
```bash
git clone https://github.com/niyatichaurasia/Interactive-Data-Visualization-Dashboard.git
cd Interactive-Data-Visualization-Dashboard
```
### 2. Install Dependencies
```bash
pip install -r requirements.txt
```

### 3. Run the App
```bash
streamlit run app.py
```
